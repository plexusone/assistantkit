# Release Notes: v0.5.0

**Release Date:** 2026-01-18

## Highlights

This release completes the module rename from `aiassistkit` to `assistantkit`.

## Breaking Changes

### Module Path Rename

The module path has been renamed from `github.com/agentplexus/aiassistkit` to `github.com/plexusone/assistantkit`.

**Migration:**

```go
// Before
import "github.com/agentplexus/aiassistkit/mcp"
import "github.com/agentplexus/aiassistkit/hooks"

// After
import "github.com/plexusone/assistantkit/mcp"
import "github.com/plexusone/assistantkit/hooks"
```

### Package Name Change

The root package name has changed from `aiassistkit` to `assistantkit`.

```go
// Before
import aiassistkit "github.com/agentplexus/aiassistkit"
fmt.Println(aiassistkit.Version)

// After
import assistantkit "github.com/plexusone/assistantkit"
fmt.Println(assistantkit.Version)
```

## What Changed

| Change | Before | After |
|--------|--------|-------|
| Module path | `github.com/agentplexus/aiassistkit` | `github.com/plexusone/assistantkit` |
| Package name | `aiassistkit` | `assistantkit` |
| Source file | `aiassistkit.go` | `assistantkit.go` |

## Installation

```bash
go get github.com/plexusone/assistantkit@v0.5.0
```
